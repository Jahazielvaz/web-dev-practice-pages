TAGS THAT SURVIVED THE CSS REVOLUTION

<em> and <i> both italizise the text they're applied to. Example:

  • This sentence has been italiziced using <em>

  • While this sentence has been Italiziced using the <i> element

<bold> and <strong> both make the text bold. Example:
  • This sentence has been made bold using the <b> element

  • While this sentence has been made bold using the <strong> element

The above mentioned elements are considered to be styling and semantic elements.

Both the <i> and the <b> tags are style tags.
Both the <em> and the <strong> tags are considered semantic.
The difference is that the style tags only serve the purpose of styling the text, while the semantic tags drive the point accross. This matters, specially in voice browsers and screen readers, which rely on translating written language into words.

In HTML5 the 2 style tags mentioned above were brought back, but this time with some semantic meaning of their own; however, the meaning is different. In order to save the redundance, it's best if you read the article here for more details and rules concerning these four tags.
