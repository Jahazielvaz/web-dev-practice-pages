ATTRIBUTES


ATTRIBUTE DEFINITION:

Attributes are used in tags to further define the tag. Attributes work on a name value structure. The only exception to this, are the "boolean attributes." These attributes have only two types of values- true of false. But instead of writing "true" or "false" for its value, you add the attribute name ot indicate true and omit it to indicate false. An exapmle is the reversed attribute in an ordered list <ol>. Adding this attribute is an indication that the list order should be reversed. We're including a link of the full attribute list here, for ease of access.


Global Attributes:

Can be applied to all elements.

Non-Global Attributes

Can only be applied to specific elements.

Global List One

  • <any element class=> = This attribute generates a selector,in order to style such selections by using CSS. you can create and use as many as you want
  • <any element id=> = It works similarl to the class attribute, how ever, there are a few differences: You can only use one of it's kind in the entire page, and the id selector will override all class selectors.
  • < any element accesskey= list of key labels> = A key label or list of key labels with which to associate the element; each key label represents a keyboard shortcut which UAs can use to activate the element or give focus to the element. An ordered set of unique space-separated tokens, each of which must be exactly one Unicode code point in length.
  • <any element contenteditable=truefalse or empty string> = Specifies whether the contents of the element are editable.
  • < contextmenu ID reference=> = The value of the id attribute on the menu with which to associate the element as a context menu.
